== Description ==

Price Robot for WooCommerce plugin calculates optimal price for products in WooCommerce. Can work as advisor or in automatic mode.

== Installation ==

1. Upload the entire 'price-robot-for-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Check settings in WooCommerce > Settings > Price Robot, and on WooCommerce admin products edit page.
